# Shopping Cart (Vue)

A Vue app that manages a multi-currency shopping cart with VAT and shipping, powered by [Dinero.js](https://dinerojs.com).

## What you'll learn

- Creating monetary values with `dinero`
- Multiplying amounts with `multiply` for quantity pricing
- Summing values with `add` for cart subtotals
- Computing VAT with `allocate` for precise tax distribution
- Converting currencies with `convert` and scaled exchange rates
- Formatting money for display with `toDecimal`

## Getting started

1. Install dependencies from the **repository root**:

   ```sh
   npm install
   ```

2. Navigate to the example:

   ```sh
   cd examples/cart-vue
   ```

3. Start the dev server:

   ```sh
   npm run dev
   ```

Check out the [Dinero.js documentation](https://dinerojs.com) to explore the full API.
