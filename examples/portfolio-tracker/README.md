# Portfolio Tracker

A React app that tracks multi-currency investment holdings and converts them to a base currency, powered by [Dinero.js](https://dinerojs.com).

## What you'll learn

- Creating monetary values with `dinero`
- Multiplying amounts with `multiply` for holding valuations
- Summing values with `add` for portfolio totals
- Converting currencies with `convert` and scaled exchange rates
- Formatting money for display with `toDecimal`

## Getting started

1. Install dependencies from the **repository root**:

   ```sh
   npm install
   ```

2. Navigate to the example:

   ```sh
   cd examples/portfolio-tracker
   ```

3. Start the dev server:

   ```sh
   npm run dev
   ```

## Learn more

Check out the [Dinero.js documentation](https://dinerojs.com) to explore the full API.
