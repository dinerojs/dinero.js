# Invoice Builder

A React app that creates, calculates, and previews professional invoices with multi-currency support, powered by [Dinero.js](https://dinerojs.com).

## What you'll learn

- Creating monetary values with `dinero`
- Multiplying amounts with `multiply` for line item totals
- Summing values with `add` for subtotals
- Computing discounts and taxes with `allocate` for precise ratio-based distribution
- Subtracting amounts with `subtract` for net calculations
- Formatting money for display with `toDecimal`

## Getting started

1. Install dependencies from the **repository root**:

   ```sh
   npm install
   ```

2. Navigate to the example:

   ```sh
   cd examples/invoice-builder
   ```

3. Start the dev server:

   ```sh
   npm run dev
   ```
