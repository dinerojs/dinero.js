# Compatibility

## Browser support

Dinero.js supports all modern browsers (Chrome, Edge, Firefox, and Safari).

::: info
Internet Explorer is no longer supported. Microsoft ended IE11 support in June 2022.
:::

## Node.js

Dinero.js runs on any [active or maintenance LTS version](https://nodejs.org/about/releases/) of Node.js.
